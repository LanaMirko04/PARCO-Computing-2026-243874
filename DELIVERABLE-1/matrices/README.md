## Matrix Generation
If you wish to generate your own matrices with different sizes or densities, you can use `mmgen.py` by running the following command in your terminal:

```shell
$ python tools/memgen.py
```

or by specifying a CSV file containsing the desired parameters:

```shell
$ python tools/memgen.py -i matrices_specs.csv
```

Note that you have to install the required dependencies listed in the `requirements.txt` file before running the script. You can do this by executing:

```shell
$ pip install -r requirements.txt
```

More details about the matrix generation process can be found in the [script's documentation](../tools/README.md).

## File Format
As required by the deliverable specification, all matrices are stored in the [Matrix Market](https://math.nist.gov/MatrixMarket/formats.html) format.

The CSV file used to generate the matrices has the following structure:

| densities | rows | cols |
|-----------|------|------|
| 0.10      | 10   | 4    |
| 0.10      | 100  | 100  |
| 0.10      | 1000 | 1000 |

Where:
- `rows`: Number of rows in the matrix.
- `cols`: Number of columns in the matrix.
- `density`: Proportion of non-zero elements in the matrix (between 0 and 1).
