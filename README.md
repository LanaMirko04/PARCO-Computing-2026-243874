# PARCO-Computing-2026-243874

This repository contains the code of the two deliverable for the Parallel Computing course of the a.a. 2026/2027 at the University of Trento.

## How to clone the repository
To clone the repository, use the following command:

```bash
git clone --recurse-submodules git@github.com:LanaMirko04/PARCO-Computing-2026-243874.git
```

## Deliverable #1
The first deliverable is located in the `DELIVERABLE-1` directory. It contains the implementation of a sparse Matrix-Vector multiplication (SpMV) using the Compressed Sparse Row (CSR) format and OpenMP for parallelization.
For more information, please refer to the ![`DELIVERABLE-1/README.md`](DELIVERABLE-1/README.md) file.
